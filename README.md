## Create a new project
1. create a project: npx create-expo-app@latest --template
2. select the blank typescript project
3. projectName: yoga-guru-app
4. open the project in vsCode
5. start the project: npm start and then press i to open the screen in iPhone

## create a new repository on the command line
1. git init
2. git add README.md
3. git commit -m "first commit"
4. git branch -M main
5. git remote add origin https://github.com/sadakhatalikhan/yoga-guru-app.git
6. git push -u origin main
